# 智能停车场管理系统 python262

## 项目概述

智能停车场管理系统是一款基于Python开发的B/S架构软件，主要实现了车位信息的智能化管理，包含前台与后台多个功能模块，致力于提高停车场管理效率。

## 技术栈

- **开发语言**：Python 3.9以上
- **数据库**：MySQL 5.7（推荐）
- **前端技术**：HTML、JQuery
- **后端技术**：Python、JSP
- **开发工具**：PyCharm
- **数据库可视化工具**：Navicat

## 功能模块

### 前台系统

- **首页**：展示停车场相关信息
- **车位信息**：显示当前车位使用情况
- **车位租用**：提供车位租赁服务
- **公告管理**：查看和管理停车场公告
- **个人中心**：用户个人信息与操作记录
- **订单管理**：管理用户的租车位订单
- **租用模拟支付**：模拟支付流程，测试支付功能

### 后台系统

- **个人中心管理**：管理个人信息
- **用户管理**：管理所有注册用户
- **车位区域管理**：设置和调整车位区域
- **车位信息管理**：更新和维护车位信息
- **车位租用管理**：监控和管理车位的租用状态
- **车位退组管理**：处理车位退租请求
- **系统管理**：
  - **轮播图管理**：更新首页轮播图内容
  - **公告管理**：发布和编辑公告

## 系统角色

系统包含以下角色：

- **普通用户**：可以租用车位，查看公告，管理个人订单等
- **管理员**：管理整个停车场系统，包括用户、车位、公告、订单等

## 运行环境

- **操作系统**：Windows
- **Web服务器**：遵守B/S架构的服务器配置

部署步骤将包含但不限于以下内容：
